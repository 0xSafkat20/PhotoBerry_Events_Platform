# PhotoBerry Events Platform

## Overview

PhotoBerry Events Platform is a modern, responsive web application designed for event photography and showcasing event galleries. The platform allows users to browse event photos, access services, and contact the team, with secure authentication options.

## Features

* User authentication via email/password, Google, and Facebook.
* Role-based access control for full content and services.
* Responsive design for mobile, tablet, and desktop.
* Interactive UI with animations using Three.js and AOS.
* Toast notifications for user feedback.
* Secure form validations for signup and login.

## Technologies Used

* **Frontend:** React.js, Next.js, Tailwind CSS, Three.js, AOS
* **Backend:** Firebase Authentication & Hosting
* **Other:** JavaScript, HTML, CSS

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/0xSafkat20/PhotoBerry_Events_Platform.git
   ```
2. Navigate to the project directory:

   ```bash
   cd PhotoBerry_Events_Platform
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Add environment variables:

   * Create a `.env` file and add Firebase configuration variables.
5. Run the project locally:

   ```bash
   npm run dev
   ```

## Usage

* Sign up or log in to access full content.
* Browse event galleries.
* Contact the team through the contact form.
* Enjoy responsive design across devices.
